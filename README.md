## ğŸš€ Usage

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/ahmadghozali-xyz/ahmadghozali-xyz.github.io.git
   cd ahmadghozali-xyz.github.io
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   open index.html
   # OR use a local server
   npx serve .
   ```

3. **Start writing!**
   - Select a font from the dropdown
   - Fill in Date, Name, and Content
   - Adjust style settings as needed
   - Click **Download** to save your handwriting

## ğŸ¨ Customization

### Adding New Fonts

1. Add your `.ttf` font file to `/fonts/`
2. Update the `fonts` array in `index.html`:
   ```javascript
   { index: 15, name: 'Your Font Name', file: 'YourFont.ttf', type: 'local' }
   ```

### Changing Paper Background

Replace `/img/folio.jpg` with your preferred paper texture.
